<h1 id="product_search">
product.search
<code>(null|object $filters = null, null|array $options = null, null|string|number $store = 0)</code>
</h1>

Retrieve list of products with basic info (id, sku, type, set, name).

#### Parameters

<table class="table">
<thead><tr><th>order</th><th>description</th></tr></thead>
<tbody>
    <tr>
        <td>0</td>
        <td><ul>
        <li>null - Retrieve list of all products.</li>
        <li>object - Retrieve list of products using specified <a href="/doc/search-filters.html" title="Search Filters">filters</a>.</li>
        </ul>
        Allowed properties for filtering: "sku", "status", "availability", "visibility", "created_at", "updated_at".</td>
    </tr>
    <tr>
        <td>1</td>
        <td><ul>
        <li>null - No options will be applied.</li>
        <li>object - Apply specified <a href="/doc/search-options.html" title="Search Options">options</a>.</li>
        </ul></td>
    </tr>
    <tr>
        <td>2</td>
        <td><ul>
        <li>null - Default store will be used.</li>
        <li>number - Specified store will be used.</li>
        <li>string - Specified store will be used.</li>
        </ul></td>
    </tr>
</tbody>
</table>

#### Return Value

An array of objects. Each object will contain "<a href="#product_properties">Product Properties</a>".

#### Example Request

Get delivery for one delivery id:

```javascript
{
    "jsonrpc" : 2.0,
    "id" : 1234,
    "method" : "call",
    "params" : [
        "be1c13ed4e03f0ed7f1e4053dfff9658",
        "product.search",
        [
            {
                "sku" : { "eq" : "product2" }
            }
        ]
    ]
}
```

#### Example Response

```javascript
{
    "jsonrpc" : 2.0,
    "id" : 1234,
    "error" : null,
    "result" : [
        {
            "sku" : "product2",
            "name" : "product2",
            "barcode" : "product2",
            "goods_type" : "NORMAL",
            "status" : 1,
            "availability" : 1,
            "visibility" : 2,
            "weight" : 12.75,
            "length" : null,
            "width" : null,
            "height" : null
        }
    ]
}
```
