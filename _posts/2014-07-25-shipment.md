<h1 id="shipment_info">
shipment.info
<code>(string $shipmentIncrementId)</code>
</h1>
Retrieve shipment information.

#### Parameters

<table class="table">
<thead><tr><th>order</th><th>description</th></tr></thead>
<tbody>
    <tr>
        <td>0</td>
        <td>string - Shipment Increment ID.</td>
    </tr>
</tbody>
</table>

#### Return Value

Shipment information.

#### Example Request

```javascript
{
    "jsonrpc" : 2.0,
    "id" : 1234,
    "method" : "call",
    "params" : [
        "97ca4f4a5cdca59e9dd0fe52e28bf7cf",
        "shipment.info",
        {
            "increment_id" : {
                "eq" : "100000020"
            }
        }
    ]
}
```
