order.create `(string $merchantCode, object $items, object $address, object $info)`
====

Create new order

#### Parameters

<table class="table"><thead><tr><th>order</th><th>description</th></tr></thead>
<tbody>
    <tr>
        <td>0</td>
        <td>string - Merchant code.</td>
    </tr>
    <tr>
        <td>1</td>
        <td>object - key - product SKU and value - quantity.</td>
    </tr>
    <tr>
        <td>2</td>
        <td>object - address properties. See the list of properties in the table below.</td>
    </tr>
    <tr>
        <td>3</td>
        <td>object - additional properties. See the list of properties in the table below.</td>
    </tr>
</tbody></table>

#### Return Value

String with a new order ID.

#### Example Request

Create new order:

```javascript
{
    "jsonrpc" : 2.0,
    "id" : 1234,
    "method" : "call",
    "params" : [
        "be1c13ed4e03f0ed7f1e4053dfff9658",
        "sales_order.create",
        [
            "rsf",
            {
                "product2" : 5
            },
            {
                "firstname" : "Bill",
                "lastname" : "Gates",
                "company" : "Microsoft",
                "street1" : "11 Times Square",
                "city" : "New York",
                "region" : "NY",
                "postcode" : 10036,
                "country" : "US",
                "phone" : "212.245.2100"
            },
            {
                "order_ref" : 12345,
                "method" : "ups_03",
                "greeting" : "Greeting text here",
                "note" : "Note text here",
                "signature_required" : false,
                "overbox" : false,
                "requested_ship_date" : "2014-07-28"
            }
        ]
    ]
}
```

#### Example Response

```javascript
{
    "jsonrpc" : 2.0,
    "id" : 1234,
    "error" : null,
    "result" : 117
}
```

order.search `(null|object $filters, array $options = array(), null|string|object $fields = array())`
====

Retrieve list of orders by filters. Order data can be customized by specifying properties to retrieve.

#### Parameters

<table class="table"><thead><tr><th>order</th><th>description</th></tr></thead>
<tbody>
    <tr>
        <td>0</td>
        <td><ul>
        <li>null - Retrieve list of all orders.</li>
        <li>object - Retrieve list of orders using specified <a href="/doc/search-filters.html" title="Search Filters">filters</a>.</li>
        </ul></td>
    </tr>
    <tr>
        <td>1</td>
        <td><ul>
        <li>null - No options will be applied.</li>
        <li>object - Apply specified <a href="/doc/search-options.html" title="Search Options">options</a>.</li>
        </ul></td>
    </tr>
    <tr>
        <td>2</td>
        <td><ul>
        <li>null - Retrieve "increment_id", "order_id", "order_ref_id" properties.</li>
        <li>string '*' - Predefined list of properties to retrieve. See the table below.</li>
        <li>object - Defined list of properties to retrieve including "increment_id", "order_id", "order_ref_id" properties.</li>
        </ul></td>
    </tr>
</tbody></table>

#### Return Value

An array of arrays or an empty array if there were no matching orders.

#### Example Request

Get order status and order items for two order ids:

```javascript
{
    "jsonrpc" : 2.0,
    "id" : 1234,
    "method" : "call",
    "params" : [
        "be1c13ed4e03f0ed7f1e4053dfff9658",
        "sales_order.search",
        [
            {
                "order_id" : [
                    114, 
                    115
                ] 
            },
            [
                "status",
                "items"
            ]
        ]
    ]
}
```
